## Pro Tips

1. **Always include hex codes** — Stitch is more consistent with exact colors
2. **Name the fonts** — Stitch will try to use them (Satoshi, Cabinet Grotesk, Outfit, etc.)
3. **Include real product data** — real names, real prices, real counts
4. **2-3 reference URLs max** — more than 3 confuses the design system generation
5. **Keep prompts under 200 words** — longer prompts don't score better
6. **Dark theme for component-level designs** (cart, PDP) — scored 0.5-1.0 points higher
7. **Light theme for page-level designs** (homepage, PLP) — better scanning, more trusted
