Implementation Notes

  • • Desktop only (≥1024px). Mobile keeps existing hamburger/sheet menu.
  • • Opens on hover with ~100ms delay, closes on mouse leave with ~300ms grace period.
  • • Brand counts and product counts are pulled from getCollection at build time.
  • • The "Popular Brands" list can be sorted by product count to auto-feature biggest brands.
  • • Keyboard accessible: Tab focuses menu trigger, Enter/Space opens, Escape closes.
  • • The mega menu is a React island (client:idle) that wraps the desktop nav items.
