🔥 Game-Changers from Astro 5

Server Islands (server:defer)

Static page shell loads instantly from CDN. Dynamic widgets (cart count, user avatar, personalized recs) load independently via server request. No client JS needed for server-rendered dynamic content. Each island can have its own cache lifetime. This is perfect for product pages — static HTML for SEO, deferred server islands for cart/auth.

Actions (type-safe server functions)

Define server functions with Zod schemas, call from client or HTML forms. Progressive enhancement — forms work without JS. Perfect for: add-to-cart, newsletter signup, review submission, contact form. Replaces manual API fetch calls with type-safe RPC.

Content Layer with Custom Loaders

Build a custom Supabase loader that fetches all 734 products at build time into a typed collection. Query with getCollection('products'). Zod-validated. Incremental builds. Replaces React Query for catalog data on SSG pages.

ISR on Vercel (Incremental Static Regeneration)

Static pages auto-revalidate on a schedule (e.g., every 4 hours). On-demand invalidation via webhook when Nyehandel syncs. No full rebuild needed for price changes. Combines CDN speed with data freshness.

Built-in i18n Routing

Native locale-prefixed routing (/sv/om-oss vs /en/about). Auto-generated hreflang tags. Fallback locales. Replaces your custom LanguageContext entirely. Critical for EU nicotine pouch market (EN, SV, DE).

View Transitions

SPA-like smooth page transitions without a client router. Add <ViewTransitions /> to layout. Supports persistent elements (cart icon stays), morph animations, slide transitions. Solves the "MPA feels slow" problem.
